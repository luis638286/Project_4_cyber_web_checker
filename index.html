Project 4 – Website Security Checker

This project is a simple Python-based tool that checks a website for basic security features, including HTTPS usage, redirects, important security headers, and cookie flags.

Features

  • Detects if the website uses HTTPS
  • Checks HTTP → HTTPS redirect
  • Scans for important security headers
  • Checks cookies for Secure + HttpOnly flags
  • Fully safe and does not attack websites

How to run the script

pip install requests python security_checker.py

Then enter any URL, like:

https://example.com
